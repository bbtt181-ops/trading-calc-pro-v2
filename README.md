## תכונות עיקריות

- **שליפת ATR אוטומטית** — מ-TwelveData API בזמן אמת
- **חישוב פוזיציה חכם** — לפי % סיכון מקפיטל, עם מגבלת 10% לפוזיציה
- **3 כניסות פירמידה** — 40% / 35% / 25% מהאחזקה
- **3 יציאות מחושבות** — לפי ATR (מניה יציבה) או % (מניה תנודתית ADR≥4%)
- **עלות ממוצעת מדויקת** — מחושבת רק מכניסות שבוצעו בפועל
- **יומן טריידים** — שמירה מקומית + סנכרון ל-Google Sheets
- **סנכרון דו-כיווני** — כניסה למחשבון ← → יומן בזמן אמת
- **תמיכה בשקלים** — המרת USD/ILS אוטומטית
- **טריילינג סטופ** — Qullamaggie (MA) ו-Minervini (תמיכות טכניות)
- **ממשק סלולרי** — מותאם מלא לטלפון

## לוגיקת חישוב

### כניסות
| כניסה | מחיר | % מהאחזקה |
|-------|------|-----------|
| #1 | P | 40% |
| #2 | P + ATR | 35% |
| #3 | P + 2×ATR | 25% |

### יציאות — מניה יציבה (ADR < 4%)
| יציאה | יעד | % מהאחזקה |
|-------|-----|-----------|
| #1 | P + 3×ATR | 40% |
| #2 | P + 4.5×ATR | 30% |
| #3 | P + 22.5% | 30% |

### יציאות — מניה תנודתית (ADR ≥ 4%)
| יציאה | יעד | % מהאחזקה |
|-------|-----|-----------|
| #1 | +20% | 40% |
| #2 | +30% | 30% |
| #3 | +45% | 30% |

### חישוב מניות
```
סיכון$ = קפיטל × % סיכון
מניות = סיכון$ ÷ (כניסה − סטופ)
אם פוזיציה > MAX_POS → חישוב מחדש לפי MAX_POS
```

### סטופ
`P − 1.5×ATR` — מקסימום 7% מ-P
